# Computer infrastructure assessment

## Purpose

This assessment task is to create a workflow that automates the collection and visualisation of recent stock market data of a group of 5 tech companies: 
- Facebook (META)
- Apple (AAPL)
- Amazon (AMZN)
- Netflix (NFLX)
- Google (GOOG)
The project is designed to be run from the command line and to support both manual execution and schedule automation. 

As the project go on the focus is on creating a repeatable, scheduable, script driven process that can be excecuted consistently.
